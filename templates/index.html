Beaconport Data Importer

This tool imports case data from the Beaconport Capture.xlsx file into the database.

Click the button below to import data from the Beaconport Capture.xlsx file. This may take a few moments.

If you have already imported data, clicking the button again will re-import the data and overwrite existing records.
